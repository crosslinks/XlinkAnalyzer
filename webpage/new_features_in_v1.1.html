New features in Xlink Analyzer version 1.1
  • improved project setup interface
  • interactive docking mode
  • buttons to selectively move a particular subunit, domain or subcomplex
  • real-time update of the cross-link display and coloring
  • new interface to manage subunits, domains, and subcomplexes
  • a database of Xlink Analyzer projects for published cross-link datasets
  • an interface to defining subcomplexes
  • auto-generation of Xlink Analyzer projects based on structures
  • get_distances.py script for calculating satisfied cross-links from command line
  • several bug fixes
